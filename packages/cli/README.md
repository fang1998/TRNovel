# @ir2026/trnovel

`@ir2026/trnovel` is the npm-distributed CLI wrapper for TRNovel.

Install it globally:

```bash
npm install -g @ir2026/trnovel
```

Or run it once with `npx`:

```bash
npx @ir2026/trnovel -h
```

This package ships the `trnovel` and `trn` commands and resolves a prebuilt Rust binary through platform-specific optional dependencies.
